# Guardian Workshop Booking System

Laravel Blade + PHP + MySQL standalone booking system.

Purpose:
A SACE-endorsed CPTD workshop booking and payment system.

Features:
- Workshop information page
- Registration
- Seat allocation
- PayFast payment
- Payflex installments
- Email notifications
- Workshop dashboard
- Reports

Tech Stack:
- Laravel
- Blade
- MySQL
- Tailwind CSS
