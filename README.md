Guardian Workshop System

A full-stack workshop management platform built to streamline the registration, payment, attendance, and certification process for professional development workshops.

Overview

The Guardian Workshop System was developed to support the administration of SACE-endorsed CPTD workshops by digitising the entire attendee lifecycle.

The platform allows participants to register for workshops, manage their profiles, submit required information, track attendance, and receive certification, while providing administrators with tools to manage workshops, attendees, payments, and reporting.

Problem Statement

Managing workshop registrations manually often results in administrative overhead, fragmented records, and difficulty tracking attendance and payments.

The goal of this project was to create a centralized system that:

Simplifies workshop registration
Tracks participant attendance
Manages workshop payments
Supports certification workflows
Provides administrators with real-time reporting and management tools
